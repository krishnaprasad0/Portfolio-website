# Professional Software Engineer Portfolio

A premium, performance-driven portfolio website designed for **Krishnaprasad R**, specializing in Flutter and cross-platform mobile development.

**Live Demo:** [https://krishnaprasad0.github.io/Portfolio-website/](https://krishnaprasad0.github.io/Portfolio-website/)


## 🚀 Key Features

- **Modern Deep Navy Theme**: A professional and sleek UI with vibrant electric blue accents.
- **Interactive Hero Section**: Dynamic particle background with smooth CSS animations.
- **Gated Resume Download**: Integrated email validation system for professional resume access.
- **Live Interaction Log**: Captures and timestamps resume download requests (securely stored in browser).
- **Owner Feature**: Secret shortcut (`Shift + L`) to export the email log as a text file.
- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop viewing.

## 🛠️ Built With

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom variables, Flexbox/Grid)
- **Interactivity**: Vanilla JavaScript (ES6+)
- **Typography**: Google Fonts (Outfit & Fira Code)

## 📂 Project Structure

- `index.html`: Main website entry point.
- `styles.css`: Full design system and layout styles.
- `script.js`: Interactive elements, particles, and validation logic.
- `resume.pdf`: Professional resume file.
- `favicon.png`: Personalized circular brand icon.

## 🏁 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/krishnaprasad0/Portfolio-website.git
   ```
2. **Open the project**:
   Simply open `index.html` in any modern web browser to view the portfolio.
